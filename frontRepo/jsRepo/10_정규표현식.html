정규표현식

        특정 규칙을 가진 문자열을 검색하거나 치환할 때 사용
        정규 표식을 이용하면 입력된 문자열에 대해 특정 조건 검색이나 치환 시
        복잡한 조건을 제시할 필요 없이 간단하게 처리가 가능하다.
    
        * 정규 표현식 객체 생성

        정규 표현식 객체를 생성하기 위해서는 정규 표현식 리터럴과 RegExp 생성자 함수를 사용한다.
        리터럴을 사용하는 것이 일반적이다. (/패턴/플래그)
    
        * 플래그 문자

        플래그 문자는 정규 표현식의 검색 방식을 설정하기 위해 사용한다.
        순서와 상관없이 하나 이상의 플래그를 동시에 설정할 수 있다.

        i : 대소문자를 가리지 않고 비교한다.
        g : 전역으로 비교를 수행한다.
        m : 여러 줄의 비교를 수행한다.
    
        * 메타문자

        특정 메타 문자를 통해 정규식으로 특정 패턴을 제시할 수 있다.
핸드폰 번호 /^010-?([0-9]{4}-?([0-9]){4})$/
