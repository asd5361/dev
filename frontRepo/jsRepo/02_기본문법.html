1. 변수와 자료형

        변수 선언***************
            var         선언 여러번 가능 (덮어씌우기 가능)
            let         일반적인 변수(최초 선언)
            const       상수 (final) : 재할당 불가능
        자료형*******************
            string      문자열
            number      숫자
            boolean     참거짓
            object      객체
            function    함수
            undefined   할당 되지 않은 변수(초기화X)
        *************************
